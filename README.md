# RotateToMouse

A node that rotates its parent towards the mouse. Useful for 2D shooters

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/RotateToMouse.git
```

## Documentation
https://github.com/ChillCube/RotateToMouse/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon RotateToMouse
```

## For Developers
```bash
push-all-addons
```
